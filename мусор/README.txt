"UserID" (int): Использован int для идентификации уникальных пользователей.
"UserName" (nvarchar(50)): nvarchar использован для текстового имени пользователя с возможностью использования юникода.
"Age" (int): Целочисленный тип данных int использован для хранения возраста пользователя.
"Email" (varchar(100)): varchar использован для хранения электронной почты с заданной максимальной длиной.
"RegistrationDate" (date): date использован для хранения даты регистрации пользователя без времени.

"ProductID" (int): Использован int для идентификации уникальных продуктов.
"ProductName" (nvarchar(100)): nvarchar использован для текстового имени продукта с возможностью использования юникода.
"Price" (decimal(10, 2)): decimal использован для точного хранения цены продукта с двумя знаками после запятой.
"Description" (text): text использован для хранения описания продукта переменной длины.

"OrderID" (int): Использован int для идентификации уникальных заказов.
"OrderDate" (datetime): datetime использован для хранения даты и времени заказа.
"TotalAmount" (money): money использован для хранения общей суммы заказа с автоматическим форматированием валюты.

"OrderDetailID" (int): Использован int для идентификации уникальных деталей заказа.
"Quantity" (int): Целочисленный тип данных int использован для хранения количества продуктов в заказе.
"UnitPrice" (decimal(10, 2)): decimal использован для точного хранения цены продукта в деталях заказа.

"AddressID" (int): Использован int для идентификации уникальных адресов.
"StreetAddress" (nvarchar(100)): nvarchar использован для текстового адреса с возможностью использования юникода.
"City" (nvarchar(50)): nvarchar использован для названия города.
"State" (nvarchar(50)): nvarchar использован для названия штата.
"ZipCode" (varchar(10)): varchar использован для хранения почтового индекса с заданной максимальной длиной.

"ReviewID" (int): Использован int для идентификации уникальных отзывов.
"Rating" (tinyint): tinyint использован для хранения рейтинга отзыва от 0 до 255.
"ReviewText" (nvarchar(max)): nvarchar(max) использован для текстового содержания отзыва переменной длины.
"ReviewDate" (datetime): datetime использован для хранения даты и времени отзыва.

"LastOrderID" (int): Использован int для хранения идентификатора последнего заказа пользователя.
